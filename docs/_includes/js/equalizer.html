Equalizer

Add the attribute data-equalizer to the container and data-equalizer-element to each element.
If you want to use more equalizer you can specify a name for each by using data-equalizer="main" and data-equalizer-element="main".
Adding data-equalizer-disable="md" or data-equalizer-disable="sm" to the data-equalizer will disable the equalizer for smaller screens.
