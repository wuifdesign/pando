Shake Element

To shake an element, use $(element).pandoShake(distance, runs, speed, distanceReduce).

Value Default Description
distance 10 Distance the element is moving in px
runs 3 Runs till the animation stops
speed 100 Speed of the animation in ms
distanceReduce 0 Reduction of the distance per run in px
